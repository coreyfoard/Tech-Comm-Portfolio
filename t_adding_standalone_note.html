Adding a Standalone Note

Add a standalone note to create a note that is independent of all items in the library.

Standalone notes are notes that are note directly related to any item. Standalone notes will appear alongside any other items in your library.

  1. Click the New Note button on the top of the center pane.
  2. Select "New Standalone Note".
  3. Type text into note.
  4. Text in notes will automatically save as you type

A note will be created as an attachment to the item, and a note editor will appear in the right-hand pane.
