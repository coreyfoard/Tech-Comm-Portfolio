Tag Selector Tool

The Tag Selector Tool can help users locate all of the tags that have been assigned to the items in the currently selected collection that match the current search.

The Tag Selector Tool searchbox filters the results of searches by tag(s). Only items that have all of the selected tags will be shown in the center pane. As you apply filters, the list of tags in the tag selector will be updated to show only the tags assigned to currently visible items. The tag selector is located at the bottom of the left Zotero pane.
